## Sobre o Projeto

A aplicação permite criar, visualizar e excluir solicitações de reembolso. Também oferece a opção de enviar o recibo correspondente, pesquisar pedidos pelo nome do solicitante, navegar por páginas com resultados paginados e acessar uma página específica com os detalhes de cada solicitação.

## Tecnologias

- **Frontend:** ReactJS, Tailwind CSS, React Query

## Instalação e Execução

### Frontend

1. Clone o repositório:
```bash
git clone https://github.com/brunomat480/refund.git
cd refund
```

2. Instale as dependências:
```bash
yarn
# ou
npm install
# ou
pnpm install
```

3. Inicie a aplicação:
```bash
yarn dev
# ou
npm run dev
# ou
pnpm dev
```

A aplicação estará disponível em `http://localhost:5173`

### Backend (API)

1. Clone o repositório da API:
```bash
git clone https://github.com/brunomat480/refund-api.git
```

2. Instale as dependências:
```bash
yarn
# ou
npm install
# ou
pnpm install
```

3. Inicie o servidor:
```bash
yarn dev
# ou
npm run dev
# ou
pnpm dev
```
